# Account Switcher

### <cf>Switch between multiple Geometry Dash accounts easily!</c>

This mod allows you to store <cl>multiple accounts</c> locally and switch between them dynamically without needing to re-enter your account login.

<co>To use this mod, go to the Account Settings and under <cg>'More'</c>, you will find the new <cg>'Switch Account'</c> button. From there, you can add your current account to the list and switch between stored accounts with ease.</co>

<cy>*This won't load your account data when switching accounts, you need to load your data from the cloud manually after switching.*</c>

## <cr>Disclaimer</c>

<cr>By using this mod, you acknowledge that your **GJP2 (encrypted password)** will be used in this mod and are stored locally and used to log in when switching accounts. If you do not wish to store this password anywhere, **Do not use this mod**.</c>
